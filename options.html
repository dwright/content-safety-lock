About Content Safety Lock

Content Safety Lock is a voluntary tool designed to help adults maintain structured guardrails against adult content. It uses voluntary labels (RTA, ICRA, SafeSurf) to identify content and block rendering.

Note: Self-Lock is scoped to the current Firefox profile. Removing or disabling the extension will also remove any active self-lock. If you need enforcement that cannot be bypassed at the browser level, consider pairing this extension with Firefox enterprise policies or a separate restricted OS user account.
